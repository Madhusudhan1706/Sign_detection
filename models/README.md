# Model Files

This directory should contain the trained model file.

## Required Files

Place your trained model file here:
- `two_stream_best.h5` (555MB) - Two-stream sign language detection model

## Note

The model file is **not included** in the repository due to its large size (555MB exceeds GitHub's 100MB limit).

### Options to obtain the model:

1. **Train your own** using the dataset provided in this repository
2. **Download from release** (if available - check the [Releases](https://github.com/Madhusudhan1706/Sign_detection/releases) page)
3. **Contact the repository owner** for access to the pre-trained model

Once obtained, place `two_stream_best.h5` in this `models/` directory before running the detection script.
